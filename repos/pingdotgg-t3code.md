## What it is

T3 Code is a lightweight, web-based graphical interface designed for AI coding agents. It provides a more visual way to interact with agents like OpenAI's Codex. It is available both as a web app and a desktop application, aiming to simplify the experience of building software with AI agents.

## Problems solved

- Clunky CLI interfaces: Replaces terminal-only agent interactions with a cleaner web GUI.
- Fragmented agent tools: Aims to provide a unified interface for multiple coding agents (Codex, Claude Code).
- Setup friction: Offers a downloader for a desktop app to make getting started 'cooler' and easier.

## How to get started

Ensure you have the Codex CLI installed and authorized. Then, run the GUI using `npx t3`. Alternatively, download and install the desktop application from the GitHub Releases page for a more integrated experience.
