## What it is

Awesome AI Apps is a curated resource library featuring over 70 practical examples, tutorials, and recipes for building LLM-powered applications using various AI frameworks.

## Problems solved

- Provides a central learning hub for diverse AI frameworks (Agno, LlamaIndex, LangChain); solves the 'blank page' problem for developers starting with MCP, RAG, or multi-agent systems; offers production-ready recipes for specialized tasks like financial reasoning and voice agents.

## How to get started

Clone the repo: `git clone https://github.com/Arindam200/awesome-ai-apps.git`. Choose a project (e.g., `cd starter_ai_agents/agno_starter`), set up your `.env` with API keys, install dependencies with `uv sync` or `pip install -r requirements.txt`, and run with `python main.py`.
