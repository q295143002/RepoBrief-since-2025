# ZeroLu/awesome-nanobanana-pro

### 1. Positioning
This repository is the definitive 'Prompt Bible' for the Nano Banana ecosystem. It targets digital artists, marketers, and AI enthusiasts who want to unlock professional-grade visual outputs from Google's latest multi-modal image generation models.

### 2. Pain Points & Advantages
Most AI image prompts are simplistic and result in generic-looking art. The catalog's advantage is its 'Technical Specificity'—it includes prompts that define camera lenses (85mm f/1.4), film stock (Kodak Portra 400), and complex lighting setups (three-point studio lighting). It specialized in 'Structured JSON Prompts' which Nano Banana parses for higher accuracy.

### 3. Technical Architecture
Managed as a community-curated Markdown collection. It categorizes prompts by functional intent: Photorealism, Creative (Droste effect/Dioramas), E-commerce (Product shots), and Workplace/Productivity (Professional headshots).

### 4. Core Implementation
Highlights include 'Character Consistency' templates that maintain facial features across different scenes, 'Split-View' 3D render prompts for product visualization, and a specialized 'Cinematic Keyframe Generator' role that turns a single image into a multi-shot storyboard description.

### 5. Code Organization
Organized by use-case chapters (1-12). Each entry features the prompt source, a visual preview of the output, and the exact text/JSON block. It includes a section on 'Education & Knowledge' for training the model on specific domain-specific visual styles.

### 6. Quick Start Suggestion
Try the 'Business Photo' prompt (Section 1.5). Upload a casual selfie and use the provided Sony A7III configuration—it demonstrates the model's ability to transform a basic input into a studio-grade headshot by defining specialized lighting vectors.

### 7. Summary
Awesome Nano Banana Pro is a masterclass in modern prompt engineering. It bridges the gap between 'casual AI use' and 'professional digital production', providing a vital resource for anyone weaponizing the latest generation of visual AI.
