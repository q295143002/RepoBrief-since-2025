## What it is

A community-curated awesome list of over 100 Claude Code skills (SKILL.md files) spanning document processing, development automation, data analysis, scientific tools, media, health, security, and project management.

## Problems solved

- Serves as a central discovery hub for the rapidly growing Claude skills ecosystem
- saving developers time in finding quality
- ready-to-use skills for their Claude Code agent workflows rather than building from scratch.

## How to get started

Browse the categorized list, click on any skill to go to its GitHub repository, and follow each skill's SKILL.md instructions to install it into your Claude Code project (typically by copying the file to `.claude/skills/`).
