## What it is

A Claude Code / Claude Cowork plugin marketplace with 65 product management skills and 36 chained workflow commands across 8 plugins — encoding frameworks from Teresa Torres, Marty Cagan, Alberto Savoia, and others — for discovery, strategy, PRDs, OKRs, roadmaps, GTM, analytics, and PM utilities.

## Problems solved

- Gives product managers structured
- framework-backed AI workflows (not generic LLM chat) to make better product decisions
- replacing reading PM books with acting on their principles via typed commands like `/discover`
- `/write-prd`
- `/plan-launch`
- `/north-star`.

## How to get started

In Claude Cowork: Customize → Browse plugins → Personal → Add marketplace from GitHub → `phuryn/pm-skills`. In Claude Code CLI: `claude plugin marketplace add phuryn/pm-skills` then install individual plugins. Skills also work in Gemini CLI, Cursor, OpenCode, Codex CLI, and Kiro via copy-paste.
