## What it is

The official reference library for developers to build ultra-low latency, multi-modal voice agents using OpenAI's Realtime WebSocket API.

## Problems solved

- High latency in traditional Turn-Based voice systems (ASR -> LLM -> TTS); complexity of managing WebSocket state for live audio; brittle tool-calling in voice-first interfaces.

## How to get started

Requires Node.js 18+ and an OpenAI API key; run 'npm install'; navigate to examples/ (e.g., node-console); run 'npm run dev'; speak into system microphone.
