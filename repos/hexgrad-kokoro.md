## What it is

A revolutionary, lightweight (82M parameters) TTS model that achieves state-of-the-art vocal quality and speed, deployable on almost any hardware.

## Problems solved

- High compute cost of quality TTS; slow generation speeds on CPU; restrictive licenses for high-end speech models.

## How to get started

Install via 'pip install kokoro'; requires 'espeak-ng'; use the KPipeline API: 'pipeline = KPipeline(lang_code="a"); generator = pipeline(text, voice="af_heart")'.
