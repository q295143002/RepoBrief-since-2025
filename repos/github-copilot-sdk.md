## What it is

A programmable SDK (for Python, TypeScript, Go, and .NET) exposing the production-tested Copilot CLI engine, allowing developers to embed Copilot's agentic workflows into custom apps.

## Problems solved

- Eliminates the need to construct custom agent orchestrators. The SDK handles planning
- tool invocations
- and file edits while letting the developer simply define the necessary agents and skills.

## How to get started

After installing the Copilot CLI, install the corresponding package (e.g., `npm install @github/copilot-sdk` or `pip install github-copilot-sdk`). Interact with the SDK client to seamlessly invoke server-mode agents.
