# liyupi/ai-guide

### 1. Positioning
This repository serves as the definitive Chinese-language encyclopedia for the 'Vibe Coding' movement. It targets students, designers, and entrepreneurs who want to leverage AI to build software without necessarily spending years learning traditional computer science.

### 2. Pain Points & Advantages
Most AI tutorials are either too shallow or too academic. Yupis Advantage is its 'Full-Stack lifecycle' approach—covering everything from the first prompt to SEO optimization and payment gateway integration. It is specifically optimized for efficiency, helping users build their first app in under 10 minutes.

### 3. Technical Architecture
Managed as a structured Markdown repository and served through a modern docusaurus-like web interface. It categorizes knowledge into concepts (AI Hub), tools (Cursor/Claude), and commercialization (SEO/Marketing).

### 4. Core Implementation
The library includes deep dives into the 'MCP' ecosystem, local model deployment (Ollama), and 'Conversation Engineering' (how to manage long LLM contexts). It also provides a registry of open-source tools and prompt templates for various business scenarios.

### 5. Code Organization
The repo is organized by learning stage: `00-Introduction`, `10-Tools`, `20-Projects`, `30-Skills`, and `50-Monetization`, making it easy to follow as a curriculum or use as a reference manual.

### 6. Quick Start Suggestion
Start with the 'Vibe Coding 10-Minute Guide'. It walks you through selecting an idea and generating a working web product using only natural language, demonstrating the power of the modern AI toolchain.

### 7. Summary
Yupi's AI Guide is a vital community utility. By distilling the chaotic AI landscape into actionable Chinese tutorials, it empowers a whole new generation of 'AI Creators' to build and ship software with unprecedented speed.
