## What it is

XCodeReviewer (formerly DeepAudit) is a sophisticated code auditing platform powered by large language models. It provides deep, intention-aware analysis of codebases, identifying security vulnerabilities, performance bottlenecks, and maintainability issues that traditional static analysis tools often miss.

## Problems solved

- Rigid rules and high false-positive rates of traditional static analysis (SAST) tools
- High cost and time required for manual code reviews in large teams
- Vague error messages (XCodeReviewer provides 'What-Why-How' fix suggestions)

## How to get started

Deploy via Vercel for a cloud-ready experience or use Docker for local hosting. Connect your GitHub/GitLab token to scan repositories. The platform supports over 10 LLM providers including Gemini, Claude, OpenAI, and local Ollama instances.
