## What it is

AstronRPA is an enterprise-grade, open-source RPA platform by iFlytek with 300+ pre-built components covering GUI, browser, Excel, PDF, vision, email, and API automation. It features a visual designer, bi-directional AI agent integration (Astron Agent), MCP services, and Docker-based server deployment.

## Problems solved

- Provides a comprehensive
- open-source alternative to commercial RPA tools (like UIPath
- Automation Anywhere) that integrates natively with AI agent workflows
- enabling both no-code automation for business users and extensible component development for engineers.

## How to get started

Server: `docker compose up -d` in the `docker/` directory. Client: Download the Release binary or build with `build.bat --python-exe [path]`. Configure `resources/conf.yaml` to point to your server. Use the visual designer to build workflows by drag-and-drop of 300+ component atoms.
