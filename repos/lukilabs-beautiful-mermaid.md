## What it is

A high-performance, dependency-free Mermaid diagram renderer for TypeScript/JS with SVG and ASCII support.

## Problems solved

- Heavy DOM dependencies in official Mermaid; lack of terminal support for diagrams; rigid theming systems; UI flickering during async rendering.

## How to get started

Install via 'npm install beautiful-mermaid'; use 'renderMermaidSVG' for browser/PDF or 'renderMermaidASCII' for terminal/logs.
