## What it is

A high-quality repository of curated system prompts and AI model configuration secrets.

## Problems solved

- Lack of transparency in commercial AI system instructions.
- Difficulty for developers to craft effective, safe, and robust system prompts for their own apps.
- Need for a centralized reference for AI behavior benchmarks.

## How to get started

git clone https://github.com/x1xhlol/system-prompts-and-models-of-ai-tools
# Browse the 'prompts/' directory.

## Detailed report

### 1. Positioning
This project is a curated collection focusing on AI system prompts and model descriptions. It targets AI developers, security researchers, and prompt engineers, aiming to reveal the underlying instruction logic of top-tier AI products like Claude and GPT-4o.

### 2. Pain Points & Advantages
- **Pain Points Solved**: Breaks the 'black box' of commercial AI instructions and provides high-quality templates for developers writing complex prompts.
- **Differentiation**: Unlike scattered social media info, this repository provides the highest signal-to-noise ratio through rigorous classification and version tracking.

### 3. Technical Architecture
- **Tech Stack**: Pure Markdown dataset with automated scripts for content collection and deduplication.
- **Design Strategy**: Uses a 'reverse engineering' perspective to showcase the evolution of AI instruction strategies over time.

### 4. Core Implementation
- **Key Content**: The `prompts/` directory covers built-in rules for mainstream models, including safety boundaries and output format specifications.
- **Key Takeaway**: Learn how to use XML tags and Markdown hierarchies to enhance model adherence to complex instructions.

### 5. Code Organization
- `/prompts`: Core instruction library, categorized by model.
- `/models`: Technical specifications for major models.

### 6. Quick Start Suggestion
Read `prompts/anthropic/claude-3-5-sonnet.md` directly; it is a perfect starting point for understanding modern long-context instruction design.

### 7. Summary
**Essence**: The 'ultimate manual' for AI developers, revealing the highest level of prompt engineering. The most notable takeaway is the structural deconstruction of complex logic into instructions.
