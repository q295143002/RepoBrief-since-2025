## What it is

A modern, enterprise-grade e-commerce backend built with Go, focusing on high concurrency, multi-tenant architectural support, and scalability.

## Problems solved

- Poor performance of traditional PHP-based e-commerce at scale; lack of robust multi-tenancy support for SaaS marketplaces; complex deployment of large-scale retail backends.

## How to get started

Requires Go 1.22+, MySQL, and Redis; clone the repo; run 'go mod download'; configure config.yaml; run 'go run main.go'; access API documentation via Swagger.
