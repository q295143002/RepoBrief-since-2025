## What it is

Sub2API is a sophisticated AI API gateway platform tailored for distributing and managing high-value AI subscriptions (such as Claude Code or Antigravity). It allows organizations or individual developers to pool multiple AI accounts and provide access via a single, OpenAI-compatible API with built-in billing, load balancing, and concurrency control.

## Problems solved

- Inefficient manual sharing of AI product subscriptions within teams
- Lack of token-level usage tracking and cost calculation for shared AI accounts
- Difficulty in building a stable API layer on top of 'per-account' rate-limited AI services

## How to get started

Deploy via the one-click installation script or Docker Compose. Use the admin dashboard to add upstream accounts (API Keys or OAuth). Users then connect using platform-generated keys. It features dedicated endpoints for specialized services like Antigravity Claude and Gemini.
