## What it is

Roboflow's optimized implementation of the DETR object detection model for edge and mobile deployment.

## Problems solved

- High latency of standard DETR models on real-time video streams
- Difficulties in deploying transformer-based vision models to small devices
- Complexity in fine-tuning object detectors for custom labels

## How to get started

Download the weights and use the Roboflow SDK to deploy your vision application.
