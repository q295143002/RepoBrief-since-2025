## What it is

A 'Meta-Framework' for Claude Code that injects elite engineering methodologies and personas into the agent's brain.

## Problems solved

- AI agents following 'Average' patterns instead of elite engineering standards
- Agent confusion when switching architectural styles
- Lack of structural 'Thinking Steps' in default agent responses

## How to get started

Clone the repo, copy profiles to `~/.claudecode/profiles`, and run `claudecode --profile senior-architect`.

## Detailed report

### 1. Positioning
SuperClaude is a 'Brain-Enhancement' framework for Claude Code CLI, providing 'Cognitive Profiles' and 'Elite Directives'.

### 2. Pain Points & Advantages
Avoids the 'Mediocrity Trap' by enforcing TDD and Clean Architecture. Includes 'Architectural Guardrails' for self-correction.

### 3. Technical Architecture
JSON Schema and advanced prompt engineering. Merges universal and project-specific rules into the system prompt.

### 4. Core Implementation
Methodology injector forces the use of specific TUI patterns and documentation standards (C4 models).

### 5. Code Organization
`/methodologies` for coding styles; `/personas` for specialized engineering roles.

### 6. Quick Start Suggestion
Activate the `solid-engineer` profile to see Claude refuse 'spaghetti code' in favor of decoupled interfaces.

### 7. Summary
The 'Engineering Senior' for AI, ensuring structural enforcement of high-level standards via prompt architecture.
