## What it is

ClawRouter is an intelligent LLM gateway that automatically routes AI requests to the best-performing model at the lowest price. Specifically optimized for autonomous agents, it uses a 15-dimension scoring system and supports non-custodial micropayments via the x402 protocol.

## Problems solved

- High costs of using flagship models (like Claude Opus) for simple tasks
- API key management overhead for multi-provider agent systems
- Latency caused by cloud-based routers (ClawRouter is <1ms local)

## How to get started

Install via curl or npm. Fund your auto-generated wallet with USDC on Base or Solana. Set it as your default model in OpenClaw, and use '/model auto' to let the router handle the trade-offs between quality and cost.
