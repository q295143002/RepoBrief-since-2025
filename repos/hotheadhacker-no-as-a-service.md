## What it is

No-as-a-Service (NaaS) is a fun, utility API that provides witty ways to say 'no'. It features over 1,000 universal rejection reasons categorized for personal, professional, and developer contexts. Built with Express, it's designed to be easily integrated into bots, landing pages, or Slack commands to provide a lighthearted way to decline requests.

## Problems solved

- Social awkwardness: Provides a humorous buffer when saying no.
- Boring UI/Bot responses: Adds personality to error messages or rejection flows.
- Developer burnout: A playful tool for declining feature requests or meetings.

## How to get started

Access the public API at `https://naas.isalman.dev/no` via a GET request. For self-hosting, clone the repo, run `npm install`, and `npm start`. It will serve random reasons from `reasons.json` on port 3000. It also has community-built extensions for Raycast, Slack, Signal, and GNOME.
