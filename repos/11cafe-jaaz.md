## What it is

Jaaz.app is an infinite canvas platform where AI agents help users create visual content. Users can 'build' ideas like playing with LEGO—sketching, pointing with arrows, and chatting with an agent to instantly generate images, videos, and storyboards. It focuses on privacy (local-first) and can use both cloud APIs (GPT-4o, Kling) and local models (ComfyUI, Ollama).

## Problems solved

- Creative blocks: Generates full images/videos from single prompts or simple sketches.
- Complex tool sets: Replaces intricate UI controls with natural language interaction and a 'magic canvas'.
- Character inconsistency: Feature built-in logic to maintain the same character across different generated scenes.
- Privacy concerns: Offers an offline version where project data never leaves the user's device.

## How to get started

Download the desktop app for Windows or macOS. For developers, clone the repo and run the React frontend and Python backend. Users can then start a project, draw on the canvas, and use the 'AI Agent' sidebar to generate or modify content dynamically.
