## What it is

ClawHub is a public registry for publishing, versioning, and discovering text-based agent skills (SKILL.md) and system lore (SOUL.md). It features a CLI-friendly API and vector-based search for easy integration.

## Problems solved

- Fragmented agent skill ecosystem
- Difficulty in sharing and updating agent behaviors
- Inefficient keyword-based skill discovery

## How to get started

Use the `clawhub` CLI to search, install, and update skills. Developers can publish their own skills by providing a `SKILL.md` file and any supporting configuration.
