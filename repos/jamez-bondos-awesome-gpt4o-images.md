## What it is

A curated awesome list of images and prompt examples generated by OpenAI's GPT-4o and gpt-image-1, demonstrating state-of-the-art AI image creation.

## Problems solved

- Helps users discover
- learn
- and apply advanced prompting techniques for GPT-4o
- providing inspiring use cases ranging from surreal compositing to product designs and cartoon illustrations.

## How to get started

Browse the categorized cases in the README. Each case provides the reference output alongside the exact Chinese/English prompts used, which you can adapt by swapping out variables like subject, style, and object.
