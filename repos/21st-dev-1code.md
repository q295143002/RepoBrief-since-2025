## What it is

1Code is a powerful, open-source desktop client for running AI coding agents. It provides a visual interface for tools like Claude Code and Codex, featuring 'Cursor-like' diff previews, git worktree isolation for safe experimentation, and the ability to run agents in persistent cloud sandboxes while the user is offline.

## Problems solved

- Fragmented CLI interfaces for different AI coding agents
- Risk of agents accidentally corrupting the main branch or local environment
- Inability to monitor or continue long-running agent tasks when a laptop is closed

## How to get started

Download the pre-built application or build from source using Bun. Once launched, connect your local repositories. You can start chat sessions that automatically create isolated git worktrees, allowing the AI to work on features or fixes without interrupting your main dev flow.
