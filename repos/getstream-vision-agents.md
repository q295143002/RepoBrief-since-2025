## What it is

Vision Agents is an open-source Python framework by GetStream for building real-time, low-latency video and voice AI agents, combining YOLO/Roboflow vision models with Gemini, OpenAI, and Claude via Stream's WebRTC edge network.

## Problems solved

- Simplifies the complex infrastructure required to deploy real-time video AI (sub-30ms latency
- WebRTC transport
- voice activity detection
- turn detection
- and model orchestration) into a clean
- composable Agent + Processor API.

## How to get started

Install via `uv add vision-agents`. Get a free Stream API key. Define an `Agent` with your chosen LLM (Gemini, OpenAI, Claude) and optional `processors` (YOLO, Roboflow). Run it to join video calls and process frames in real time.
