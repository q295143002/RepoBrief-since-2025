# charmbracelet/crush

### 1. Positioning
Crush is a 'Luxury' terminal coding agent powered by Charm's Bubble Tea, designed for efficiency and aesthetic pleasure.

### 2. Pain Points & Advantages
Reduces the 'Feedback Gap' with vibrant TUI animations. Offers pixel-perfect diff visualization within the terminal.

### 3. Technical Architecture
Go based using Bubble Tea (TUI) and Lip Gloss (Styling). Adopts the MVU pattern for smooth streaming UI.

### 4. Core Implementation
High-reliability executor manages LLM connections and shell commands. Features real-time, color-coded Git-like diffs.

### 5. Code Organization
`/ui` for visual components; `/pkg/agent` for context and tool logic.

### 6. Quick Start Suggestion
Run a project explanation task to witness the most beautiful summary ever rendered in a terminal.

### 7. Summary
A 'Senior Architect' with a design degree, leveraging high-end TUI for superior human-agent collaboration.
