## What it is

A high-performance, open-source screen recording application for macOS built with Swift and React, offering seamless sharing and editing.

## Problems solved

- Slow and clunky default screen recording tools; difficulty sharing recorded clips instantly via URL; limited editing capabilities in lightweight recorders.

## How to get started

Download the DMG from the website or GitHub releases; grant screen recording and accessibility permissions; use the menu bar icon or shortcuts to start recording; upload to Cap Cloud for instant sharing.
