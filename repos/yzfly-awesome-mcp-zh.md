## What it is

Awesome-MCP-ZH is a curated list of resources for the Model Context Protocol (MCP) specifically tailored for Chinese users. It includes comprehensive documentation, a directory of popular MCP clients (like Claude Desktop, Cherry Studio, and Cursor), and an extensive collection of categorized MCP servers for browsing, coding, databases, and more.

## Problems solved

- Information fragmentation: Centralizes various MCP tools and servers in one place.
- Language barrier: Provides Chinese explanations and guides for the MCP ecosystem.
- Tool discovery: Lists specialized MCP servers for local Chinese services (e.g., Bilibili, WeChat, Douyin, Aliyun).
- Beginner entry: Offers simple setup guides for non-developers.

## How to get started

Browse the categorized lists in the README to find suitable MCP clients and servers. Follow the links to individual repositories for installation instructions. Use the provided articles to understand the MCP architecture and use cases.
