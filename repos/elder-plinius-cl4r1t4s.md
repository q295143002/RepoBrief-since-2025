# elder-plinius/CL4R1T4S

### 1. Positioning
CL4R1T4S is an adversarial transparency project providing a repository of reverse-engineered system prompts and operational parameters utilized by large AI service providers.

### 2. Pain Points & Advantages
The opaque nature of commercial AI systems obscures their ethical frames and operational heuristics. This repo empowers users and researchers to understand the actual instructions governing an AI's behavior.

### 3. Technical Architecture
It is purely a data repository (mostly text and markdown). It does not contain executable architecture but serves as an OSINT (Open Source Intelligence) knowledge base.

### 4. Core Implementation
Contributors use specific injection or “jailbreak” strategies to compel AI systems to output their verbatim system prompts, appending these as documented evidence in the repository.

### 5. Code Organization
Directories are typically sorted by AI provider (e.g., OpenAI, Anthropic, Google) and subsequently by product feature or model version.

### 6. Quick Start Suggestion
Read through the Anthropic or OpenAI folders to observe how major organizations construct rigorous behavioral frameworks using extensive meta-prompting techniques.

### 7. Summary
A fascinating, community-driven transparency vault that demystifies modern AI operations by exposing the underlying rules and system instructions guiding popular language models.
