## What it is

A high-performance Linux compatibility layer for running Windows applications with AI-driven optimization.

## Problems solved

- Incompatibility of mission-critical Windows software on Linux workstations
- Complexity and performance overhead of traditional Wine/Proton configurations
- Manual 'Trial-and-Error' required to fix registry issues and DLL conflicts

## How to get started

Clone the repo and run `./install.sh`. Launch apps using `winboat run app.exe`. The AI agent will auto-configure the prefix for you.
