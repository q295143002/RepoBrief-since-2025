## What it is

mcp-use is a comprehensive full-stack framework for the Model Context Protocol (MCP), enabling developers to build, test, and deploy interactive MCP servers and UI-driven 'MCP Apps'.

## Problems solved

- Standardizes the bridge between AI agents and local/hosted tools; solves the 'no UI' problem of standard MCP by introducing React-based widgets; provides a unified SDK for both TypeScript and Python
- reducing the learning curve for multi-language teams.

## How to get started

Start a new project with `npx create-mcp-use-app@latest`. For Python, install via `pip install mcp-use`. Use the built-in inspector at `server.listen(3000)` to debug your tools and widgets in the browser instantly.
