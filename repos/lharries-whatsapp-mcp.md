# lharries/whatsapp-mcp

### 1. Positioning
lharries/whatsapp-mcp is positioned as a this mcp (model context protocol) server bridges personal whatsapp accounts with ai agents like claude. it allows agents to search contacts, read message history (including media), and send messages/files directly from the llm interface. It aims to provide developers with a robust, Go-based solution for whatsapp mcp server.
