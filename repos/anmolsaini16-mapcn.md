## What it is

mapcn is a collection of React components for integrating powerful maps into web applications. Built on MapLibre GL and styled with Tailwind CSS, it follows the design patterns of shadcn/ui. The components are theme-aware, composable, and come with sensible defaults for markers, popups, and routes, requiring 'zero config' for basic setups.

## Problems solved

- Complex map integration: Simplifies MapLibre GL setup into declarative React components.
- Design inconsistency: Adapts automatically to light/dark themes and matches shadcn/ui aesthetics.
- Boilerplate code: Provides one-command setup for common features like markers, zoom controls, and fullscreen modes.
- Basemap licensing confusion: Clearly defines terms for CARTO Basemaps and provides alternatives.

## How to get started

Install via npx or your preferred package manager. Import components like `MapContainer`, `Marker`, or `Route` into your React project. The maps work out of the box with default styles, but can be fully customized using Tailwind or MapLibre's underlying API.
