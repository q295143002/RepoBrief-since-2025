## What it is

HexStrike AI is an MCP-based cybersecurity automation platform integrating 150+ security tools (Nmap, SQLMap, Nuclei, Ghidra, etc.) with 12 specialized AI agents for automated penetration testing, CTF solving, and bug bounty hunting.

## Problems solved

- Dramatically accelerates security assessments (up to 144x faster than manual testing) by automating subdomain enumeration
- vulnerability scanning
- web app testing
- binary analysis
- and reporting through AI-orchestrated tool chains.

## How to get started

Clone the repo, install Python deps, and start the MCP server (`python3 hexstrike_server.py`). Configure Claude Desktop, VS Code Copilot, Cursor, or Roo Code to use the MCP. Prompt the AI to run specific security assessments on authorized targets.
