## What it is

A single CLAUDE.md (or Claude Code plugin) encoding four Karpathy-derived principles — Think Before Coding, Simplicity First, Surgical Changes, and Goal-Driven Execution — to fix common LLM coding anti-patterns.

## Problems solved

- Addresses LLMs silently making wrong assumptions
- overcomplicating code with bloated abstractions
- making unasked-for side-effect changes
- and lacking verifiable success criteria
- directly based on Andrej Karpathy's published criticisms of AI coders.

## How to get started

Install as a Claude Code plugin: `/plugin marketplace add forrestchang/andrej-karpathy-skills`, then `/plugin install andrej-karpathy-skills@karpathy-skills`. Or copy-paste the `CLAUDE.md` file directly into any project or append it to an existing one.
